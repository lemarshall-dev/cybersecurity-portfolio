# Computer Fundamentals

## Objectives
- Understand the fundamentals of different type of computers and how they interact with eachother.
- Understand the client-server model.
- Understand the following concepts on a surface level:
  - DNS
  - Client
  - Server
  - Port
  - Protocol
  - Network

## What I learned
- HTTP (Hypertext transfer Protocol) is a stateless client-server protocol meaning that each request is processed independantly, retaining no information about previous requests
- In order for information to be stored, session identifiers need to be created, which are then stored in cookies or tokens.
- There are 9 core HTTP methods:
  - GET
  - POST
  - PUT
  - DELETE
  - PATCH
  - HEAD
  - OPTIONS
  - CONNECT
  - TRACE
