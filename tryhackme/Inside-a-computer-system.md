## What did I learn
- BIOS has been mostly replaced bu UEFI (Unified Extensible Firmware Interface); it does the same job, just a different name.
- UEFI (or older BIOS) firmware initializes and coordinates components. It's the first software that runs.
- UEFI runs POST to check that required hardware is present, configured, and functioning. Errors trigger beeps or alerts.
- It's important to know how each component works with each other for the cybersecurity roles.
