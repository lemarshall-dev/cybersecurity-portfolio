# TryHackMe First Room Write-up

## Objective
Learn the very basic of cybersecurity concepts and navigation.

## Tools Used
- TryHackMe platform
- Basic Linux commands

## What I did
- Completed introductory room which involved hacking into a bank to transfer funds into my account. (If only!)
- Practised navigating Linux terminal with very limited commands

## What I learned
- Offensive Security is a term used to describe someone exploiting security vulnerabilities of *their* system before someone else does externally. Also known as Ethical Hacking.
  Terminal is an important and powerful tool in the cyber security space.
