## Objective

## Tools Used

- ChatGPT (General Advice)
- Safari for installs

## Environment Set-up

## Methodology
The objective of this lab was to set up a virtual machine using VMware Fusion to create a controlled environment for testing and learning.

I set up a new virtual machine and configured it using minimal resources ensuring it would function as required without wasting host machine resources.

Once the VM was running and the Ubuntu OS was installed, I updated the system to ensure it was using the latest packages and security patches.

A snapshot was created to preserve a clean state, allowing the system to be restored easily if changes caused instability or misconfiguration.

## Key findings / issues
1. Performance was sluggish after intially setting up. This drastically slowed down the install process. An increase in RAM from 2gb to 4gb was all that was required. 
2. Ubuntu GUI wasn't massively different from Windows or MacOS
3. When installing OS, WIFI option wasn't available, only a wired connection or no network were available options.

## Outcome
The VM has been set up successfully and is now available to launch whenever needed. Performace is stable and snapshots have been set up to provide a safety net.
