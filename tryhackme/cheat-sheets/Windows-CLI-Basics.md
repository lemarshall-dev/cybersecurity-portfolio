# CLI basics for Windows

## What is the command prompt
The Windows command prompt is often referred to as CMD. It's the exact same as the Linux Terminal however it does not necessarily use the same syntax.

## Command Basics

### Moving About
cd | This shows the full path of your current directory. It's the same as Linux's pwd
dir | This prints a full list of the directory. Files and folders included. It's the same as Linux's ls
dir /a | This will show hidden and visible folders and files. | This is the same as Linux's ls -a
dir /s | This makes windows search for a specific file. The /s tells windows to search all subfolders starting in my current directory. 
type (filename) | This will print out all text inside the file.

### System Information
whoami | This prints the current user_ID (Username)
hostname | This prints the name of the machine you're using
systeminfo | This prints a list of system information like hardware and OS information
ipconfig | This prints the machines network configuration
