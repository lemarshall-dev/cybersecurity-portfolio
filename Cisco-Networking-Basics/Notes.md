# Cisco Packet Tracer
A packet trace is a tool which allows you to simulate a real network. It allows you to 
- Add virtual devices and connect them via cables/wireless
- Select, delete, inspect, label and group components within your network, kind of like mapping out all the devices in your home
- Manage your network

Network management menu lets you: 
- Open an existing/sample network
- Save your current network
- Modify your user profile
